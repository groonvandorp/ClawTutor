## Countdown Rules (Configurable)

Default schedule (adjustable per exam):
- **14 days before:** First mention, start reviewing weak topics
- **7 days before:** Daily check-ins, intensified practice
- **3 days before:** Focus mode, exam-relevant topics only
- **1 day before:** Final review, confidence boost
- **Day of:** "Good luck! You've got this! 💪"

### Custom Countdown

Override per exam by adding a countdown column:
```markdown
| 2024-05-20 | Finals | Everything | countdown: 21,14,7,3,1 |
```

Or set global defaults:
```markdown
## Settings
default_countdown: 14,7,3,1
```
