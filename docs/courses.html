How every course works

📚

Curriculum-aware

Upload your textbook's table of contents. The tutor knows the chapter, the weak spots, and upcoming exams.

📊

Progress tracking

Automatic per-topic scoring: ⬜ not started → 🟡 learning → 🟢 good → ⭐ mastered. Visible in weekly reports.

🎯

Adaptive difficulty

The tutor focuses on weak areas and increases difficulty as the child improves. No wasted time on topics already mastered.

💰

Screen time credits

Every correct answer earns screen time. Works the same across all courses — one shared credit balance per child.
