Exam Prep Mode

Add an exam date and the tutor shifts gears. Countdown-based intensification with automatic weak-spot targeting.

How It Works

Register upcoming exams in CALENDAR.md. The tutor reads this file every session and automatically adjusts its approach based on how many days remain.

| Date | Subject | Topic | Status | |------------|---------|-----------------|---------------| | 20.03.2026 | Math | Fractions | 🟡 14 days | | 05.04.2026 | English | Present Perfect | ⬜ 30 days |

The Countdown

14 days

Coverage Phase

All subtopics get reviewed. The tutor makes sure nothing is missed.

7 days

Weak Spot Focus

PROGRESS.md shows which topics are below 60%. Those get extra attention.

3 days

Mixed Practice

Questions mirror the real test format — mixed topics, varied difficulty.

1 day

Mini Mock Exam

A practice test covering the full scope. Results reviewed together.

Test day

Motivation Only

"Du schaffst das! 💪" — No quiz, no pressure. Just encouragement.

Exam Detail Files

For each exam, create a detail file in exams/ with the exact scope:

# Math Exam — 20.03.2026 ## Topics Covered - Adding fractions with same denominator - Adding fractions with different denominators - Multiplying fractions - Word problems with fractions ## From the Teacher "Focus on real-world applications. Calculator NOT allowed."

The tutor reads this file and tailors its questions to match exactly what will be tested.

Automatic Integration

  • PROGRESS.md — Shows current mastery per topic
  • CURRICULUM.md — Maps topics to textbook chapters
  • CALENDAR.md — Tracks exam dates and countdown
  • exams/*.md — Detailed scope per test

When these files exist, exam prep is automatic. No extra configuration needed.
